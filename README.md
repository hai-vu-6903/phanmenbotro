## Phần mềm bổ trợ học tập cho Hạ sĩ quan – Binh sĩ
## 1.Giới thiệu

Phần mềm bổ trợ học tập là một web application được phát triển nhằm hỗ trợ chính trị viên và cán bộ quản lý trong việc giảng dạy, phổ biến kiến thức cho hạ sĩ quan – binh sĩ mới trong đơn vị.

Ứng dụng cung cấp các nội dung học tập quan trọng như bài hát truyền thống, quy định trong đơn vị, vũ điệu sinh hoạt tập thể và thông tin về đơn vị đang đóng quân, giúp bộ đội dễ dàng tiếp cận và ôn luyện kiến thức một cách trực quan.

Website được xây dựng bằng ReactJS kết hợp với Vite, tối ưu tốc độ tải trang và trải nghiệm người dùng.

🌐 Demo: https://phanmembotro.web.app/

## 2.Tính năng chính

* Danh sách bài hát quy định phục vụ sinh hoạt và học tập
* Vũ điệu sinh hoạt tập thể kèm nội dung hướng dẫn
* Các quy định cần nắm trong đơn vị
* Thông tin tìm hiểu về đơn vị đang đóng quân
* Hệ thống câu hỏi trắc nghiệm giúp ôn luyện kiến thức
* Hiển thị kết quả và tiến độ làm bài
* Giao diện đơn giản, dễ sử dụng trên trình duyệt

## 3.Công nghệ sử dụng

* **ReactJS** – Xây dựng giao diện người dùng theo mô hình component
* **Vite** – Công cụ build nhanh cho dự án frontend
* **JavaScript (ES6+)**
* **HTML5 / CSS3**
* **Firebase Hosting** – Deploy website

## 4.Cài đặt và chạy project

git clone https://github.com/hai-vu-6903/phanmenbotro.git

cd your-repository

npm install

npm run dev

Sau đó mở trình duyệt tại: http://localhost:5173

## 5.Mục đích dự án

Dự án được thực hiện nhằm:

* Hỗ trợ công tác giáo dục chính trị trong đơn vị
* Giúp hạ sĩ quan – binh sĩ mới dễ dàng học tập và ôn luyện
* Áp dụng kiến thức ReactJS và phát triển web frontend
